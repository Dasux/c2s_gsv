\chapter{Cshrc configuration code}
\label{app:csh}



\noindent The following \inlinecode{.cshrc} configuration file must be present in the user environment for proper Cadence tool initialization. Since several Cadence utilities expect a C-Shell environment, the user must first switch from the default shell to C-Shell by executing the command \inlinecode{csh} in the terminal before launching any Cadence tools. \\

\vspace{0.5in}

A standard RHEL installation does not typically create a user-level \inlinecode{.cshrc} file by default, as Bash is generally configured as the primary login shell. Therefore, if no existing \inlinecode{.cshrc} file is present in the user home directory, create one manually using the configuration provided below. If a \inlinecode{.cshrc} file already exists, its contents should be replaced or merged carefully with the required configuration.
